HTML5 Canvas javascript Lib.

This javascript library will provide you a glimpse of JavaScript OOP concept. You start by extending Canvas Ojbect.

var domElem='#output' || document.querySelector('#output canvas') || document.body
var canvas=new Canvas(domElem);

In this point you've a canvas reference in browser, where you can perform:

  • add=function(drObject) to add in canvas.
  • rem=function(drObject) to delete from canvas.
  • clear=function() to clean canvas.
  • draw=function() to show items in canvas, draw once
  • animate=function(interval) to show items in canvas, draw infinte times per 1000ms,1000/60 is default and false is to stop.
